# NetworkSimulator
### Dependencies
* Flask
..* sudo pip install flask
* Mininet
..* sudo apt-get install mininet
### Quick start
* Clone repo
..* git clone https://github.com/Sirozha1337/NetworkSimulator.git
* cd to cloned repo
..* cd NetworkSimulator 
* Run 'sudo python Builder.py'
* Open '127.0.0.1:5000' in your browser
